# firebase2pocketbase

Claude Code（Claude MCP環境）用のスキルコレクションです。

## インストール

このリポジトリをクローンし、Claude Codeのスキルディレクトリとして登録してください。

## 利用可能なスキル

| スキル名 | 説明 |
|---------|------|
| `firebase2pocketbase` | React+Firebase JSXをPocketBaseプロジェクトに変換 |

各スキルの詳細は `skills/<スキル名>/SKILL.md` を参照してください。

## 使い方

```
/<スキル名> [引数]
```

例:
```
/firebase2pocketbase path/to/App.jsx
```
